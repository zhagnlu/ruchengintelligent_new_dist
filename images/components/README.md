# Component Images for Product Showcase

This directory should contain the following images for the CNC Bending Machine components section:

## Required Images:

1. **mechanical-structure.jpg**
   - Shows the main frame, columns, ram, and worktable
   - Highlights the welded steel construction
   - Recommended size: 800x600px or higher

2. **hydraulic-system.jpg**
   - Shows hydraulic cylinders, pumps, and valves
   - Displays the hydraulic control panel
   - Recommended size: 800x600px or higher

3. **cnc-control.jpg**
   - Shows the CNC control panel with touch screen
   - Displays the operator interface
   - Recommended size: 800x600px or higher

4. **back-gauge.jpg**
   - Shows the back gauge system with positioning arms
   - Displays the ball screw mechanism
   - Recommended size: 800x600px or higher

5. **tooling-system.jpg**
   - Shows punch and die tooling
   - Displays tool holders and setup
   - Recommended size: 800x600px or higher

6. **safety-system.jpg**
   - Shows light curtains and safety guards
   - Displays emergency stop buttons
   - Recommended size: 800x600px or higher

## Image Guidelines:

- Format: JPG or PNG
- Minimum resolution: 800x600px
- Recommended resolution: 1200x900px
- Aspect ratio: 4:3 or 16:9
- File size: Optimize for web (under 500KB per image)
- Quality: High-quality product photography with good lighting
- Background: Clean, professional background (white or light gray preferred)

## Alternative:

If specific component images are not available, you can use:
- Overall machine photos highlighting each component area
- 3D renderings of components
- Technical drawings or diagrams
- Stock photos from the manufacturer
