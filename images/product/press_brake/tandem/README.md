# Tandem Press Brake Images

This directory should contain the following images for the tandem press brake product page:

## Required Images:

1. **synchronized_control.jpg** - Synchronized CNC Control System
   - 同步数控控制系统
   - Shows the master-slave CNC coordination system

2. **hydraulic_system.jpg** - Dual Hydraulic System
   - 双液压系统
   - Shows the coordinated hydraulic systems of both machines

3. **back_gauge.jpg** - Multi-Axis Back Gauge System
   - 多轴后挡料系统
   - Shows the synchronized back gauge systems

4. **crowning_system.jpg** - CNC Crowning System
   - CNC补偿系统
   - Shows the automatic crowning compensation system

5. **tooling_system.jpg** - Segmented Tooling System
   - 分段模具系统
   - Shows the segmented punch and die tooling

6. **safety_system.jpg** - Integrated Safety & Monitoring System
   - 集成安全监控系统
   - Shows the networked safety system with light curtains

## Image Specifications:
- Format: JPG
- Recommended resolution: 1200x800px or higher
- File size: Optimized for web (< 500KB per image)
