System Components

Understanding the advanced technology behind our tandem press brake synchronization system

Dual Machine Configuration

Two high-precision CNC press brakes positioned in-line and connected through advanced communication systems. Each machine maintains its own structural integrity while working in perfect coordination with its partner, enabling the bending of workpieces up to 6 meters or longer.

Master-Slave Control System

Advanced CNC coordination where one machine acts as the master controller, coordinating all movements with the slave machine. Real-time communication ensures microsecond-level synchronization, preventing any misalignment during the bending process. Both controllers can display the same program and status.

Synchronized Back Gauge System

Multi-axis back gauge systems on both machines communicate and coordinate their positioning. Networked control ensures that both back gauges move to identical positions simultaneously, maintaining perfect workpiece alignment across the entire length. Features automatic error detection and correction.

High-Speed Communication Network

Dedicated high-speed network connection between the two machines enables real-time data exchange. Continuously monitors position, speed, force, and angle, making instant adjustments to maintain synchronization. Includes redundant communication paths for fail-safe operation.

Coordinated Hydraulic System

Each machine has its own independent hydraulic system, but both are coordinated through the master control. Pressure and speed are synchronized to ensure uniform bending force distribution across the entire workpiece length. Advanced servo valves provide precise control and instant response.

CNC Crowning System

Advanced CNC crowning compensation system automatically adjusts for beam deflection across the extended working length. The system calculates optimal crowning values based on material thickness, length, and bending force, ensuring consistent bend angles from end to end. With four cylinders working in coordination, the crowning system maintains perfect straightness even when processing extra-long workpieces, eliminating the common problem of center deflection.

Frequently Asked Questions

Common questions about tandem press brake systems

Q:What is the maximum workpiece length a tandem system can handle?

Tandem configurations can handle extra-long workpieces well beyond standard single-machine capacity. Custom configurations can be designed for various length requirements based on your specific applications.

Q:How accurate is the synchronization between the two machines?

Our tandem systems maintain exceptional synchronization accuracy throughout the bending process. Advanced sensors continuously monitor and adjust for perfect coordination between both machines.

Q:Can the machines operate independently?

Yes, each machine can be switched to independent operation mode for standard-length workpieces, effectively giving you two separate press brakes when tandem operation is not needed.

Q:What happens if one machine encounters an error?

The system includes comprehensive error handling. If either machine detects an issue, both machines automatically stop to prevent damage to the workpiece. Diagnostic information is displayed on both controllers.

Q:How long does installation and commissioning take?

Installation typically requires 2-3 weeks, including machine positioning, network setup, calibration, and synchronization testing. We provide comprehensive on-site commissioning and operator training.

Q:What is the warranty coverage?

We provide a comprehensive 2-year warranty covering all major components including the synchronization system. Extended warranty packages and dedicated support programs are available.
